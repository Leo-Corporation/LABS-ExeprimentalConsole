# LABS Experimental Console
## Introduction
LABS ExperimentalConsole est une application console dédiée à des tests.
Le programme a été réécrit en C#, et en .NET Core 3.
## Commandes
Ce programme inclus plusieurs commandes comme :
* ver
* help
* clear
* cls
* about
* list
* update
* repo
* logo
* usage
* beep
